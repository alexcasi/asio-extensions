Detailed Description

Reads the entire contents of a file into memory.

Function Documentation

template<class ByteArray >
void asioext::read_file ( const char *  filename,
ByteArray &  c 
)

Read a file into a container.

This function loads the contents of filename into c.

Parameters
filenameThe path of the file to load.
cThe container object which shall contain the file's content. The container is resized to the file's size and any previous data is overwritten. The container type must satisfy the ByteArray requirements.
Exceptions
asio::system_errorThrown on failure.

template<class MutableBufferSequence >
void asioext::read_file ( const char *  filename,
const MutableBufferSequence &  buffers 
)

Read a file into a sequence of buffers.

This function loads the contents of filename into buffers.

Parameters
filenameThe path of the file to load.
buffersThe sequence of buffers to read the file into. If the file size is less than the total size of all buffers, the call will fail. Otherwise the buffer is filled entirely. Trailing data is ignored.
Exceptions
asio::system_errorThrown on failure.
